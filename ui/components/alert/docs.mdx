Alerts

<div className="lead">
  Alerts are the Design System method of providing extra relevant information to the user and an optional related action.

Alerts are composed of a background colour, solid left edge on mobile, copy and an optional button. Alerts can be associated with specific list items, or appear separately in the alert relates to something wider.

</div>

## FYI Alert

FYI alerts are used to share non-critical information that users will still find useful. The colour of an FYI alert is the interactive colour of the chosen colour palette for that page.

<Example title="Alert Informational">
  <CodeView>
    <Alert type="info">

     <h2>
     Your bill has been consistent for the last 3 months.{' '}
      <a href="javascript:void(0);" className="alert-link">Setup Direct Debit</a>
    </h2>
    </Alert>
  </CodeView>
</Example>

## Critical Alert

Critical alerts are used to display critical information, and always appear in System Red. Critical alerts can have an action associated inside the alert, which is a Secondary button in System Red.

<Example title="Alert Warning">
  <CodeView>
    <Alert type="warning">

     <h2>
        Please pay the outstanding amount now or arrange an extension.{' '}
          <a href="javascript:void(0);"  className="alert-link">Setup Direct Debit</a>
        </h2>
    </Alert>
  </CodeView>
</Example>

## Error Inline Alert

Use the error variant when you need to alert the user about an error in the system that needs immediate attention.

<Example title="Alert Error">
  <CodeView>
    <Alert type="error">

        <h2>
       <b> Elliots Business Visa   </b> <span className="alert-accent">Card expires soon</span>
        </h2>
    </Alert>
  </CodeView>
</Example>

## Recommendation Alert

Recommendation alerts are used to notify user of a recommendation and don't have a specific action attached.

<Example title="Alert Recommendation">
  <CodeView>
    <Alert type="recommendation">

   <h2>
       <b>Monthly Pass{' '}</b>
       <p>$16.99 with 2 weeks for free.</p>
        <span  className="alert-accent">Recommended for you</span>
        </h2>
    </Alert>
  </CodeView>
</Example>
